Game Interface

Below is the interface of one of the Lost Land games:

Attention: The interface of the game can slightly vary according to the version you are playing. The below information covers default options available throughout the series.

  1. Play: Click play to display two game sections.

    1. Main game: Click OK under the main game section to start the main story of the game.

    2. Bonus Chapter: Click OK under the bonus chapter section to start the back-end story of the game. This chapter is unlocked only after completing the main game.

    Note: Click Start Over to start the game from the beginning.

  2. Option: Click option to update the below game settings:

    1. Music

    2. Environment

    3. Sound effects

    4. Voice over

    5. Change language

  3. Extras: Displays the below list of features you can access after completing the game:

    1. Videos

    2. Mini Games

    3. Puzzles

    4. Achievements

    5. Bonus Game

    6. Collections

Tip: Click Hint in the right bottom corner of the screen to display where you should go next or what move should be made next.
